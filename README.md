Instruksi Untuk Menjalankan File Project :
1. Buka NetBeans lalu buka Import Project dan Import From Zip lalu pilih SendToIn.zip.
2. Setelah File Project Berhasil di import, sekarang jalankan XAAMP dengan Apache dan MySql service di jalankan
3. Buke Aplikasi Browser apapun lalu ketikan http://localhost/phpmyadmin/ untuk memasuki halaman phpmyadmin
4. IMPORT sql database SendToIn lalu kembali ke NetBeans dan IMPORT jar mysql-connector.jar
5. Setelah Itu Jalankan Aplikasi dan Anda Akan di arahkan Langsung ke halaman Login Page.
6. Silahkan Login Berdasarkan Username dan Password yang sudah terdaftar di Database
